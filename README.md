# 지아세 오늘의 나 — 텔레그램 봇

## 1. 봇 토큰 발급
1. 텔레그램에서 @BotFather 검색 → `/newbot` → 이름 설정 → 토큰 발급
2. `.env.example`을 `.env`로 복사하고 토큰 입력

```
cp .env.example .env
```

## 2. 설치 및 실행

```
npm install
npm start
```

## 3. 사용 흐름
- `/start` — 가입 (출생연도 입력 → 힐링모드 선택)
- `/오늘` — 즉시 확인
- `/stop` — 수신 거부(데이터 즉시 삭제)
- `/privacy` — 수집 항목 안내

## 4. 매일 발송
서버가 켜져 있는 동안 매일 **07:00(한국시간)** 에 가입자 전체에게 자동 발송됩니다.
(`bot.js` 안의 `cron.schedule('0 7 * * *', ...)` 에서 시간 변경 가능)

## 5. 구조
```
bornbone_today.js  — 구성기학 "오늘의 나" 계산 엔진 (웹 랜딩과 동일 소스, 외부 의존 0)
phrases.js         — 힐링·정화 멘트 뱅크 (비노출/노출 두 톤, 유형별 4변형 로테이션)
storage.js         — 사용자 저장(JSON 파일, 소규모 운영 기준)
bot.js             — 메인 서버 (온보딩 + 일일 발송 + 위기개입 키워드 감지)
users.json         — 사용자 DB (런타임에 자동 생성/갱신)
```

## 6. 운영 규모가 커지면
- `storage.js`만 SQLite/Postgres 버전으로 교체 (다른 파일 수정 불필요)
- 일반 공개봇으로 확장 시: 개인정보처리방침 정식화, `/privacy` 문구를 실제 방침 링크로 교체 필요

## 7. 주의
- 위기개입 키워드(자살·자해 표현) 감지 시 모드·온보딩 상태와 무관하게 즉시 안내 문구로 전환됩니다.
  `bot.js`의 `CRISIS_KEYWORDS` 배열 — 임의로 삭제하지 마세요.
